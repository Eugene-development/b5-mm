# Обновление padding полей формы

## Изменения
Уменьшен padding полей формы для улучшения отображения на мобильных устройствах.

## До изменений
```css
class="p-3 sm:p-2"
```
- Мобильные: `padding: 0.75rem` (12px)
- Десктоп: `padding: 0.5rem` (8px)

## После изменений
```css
class="px-3 py-2"
```
- Все устройства: 
  - `padding-left: 0.75rem` (12px)
  - `padding-right: 0.75rem` (12px)
  - `padding-top: 0.5rem` (8px)
  - `padding-bottom: 0.5rem` (8px)

## Преимущества

1. **Лучше для мобильных** - меньшая высота полей экономит место на экране
2. **Консистентность** - одинаковый padding на всех устройствах
3. **Удобство** - горизонтальный padding остается достаточным для комфортного ввода
4. **Компактность** - форма занимает меньше места по вертикали

## Затронутые поля

- Ключ агента
- Имя клиента
- Телефон клиента
- Адрес объекта
- Комментарий (textarea)

## Визуальное сравнение

### До (мобильные)
```
┌─────────────────────────┐
│                         │  ← 12px padding сверху
│   Текст ввода           │
│                         │  ← 12px padding снизу
└─────────────────────────┘
```

### После (все устройства)
```
┌─────────────────────────┐
│                         │  ← 8px padding сверху
│   Текст ввода           │
│                         │  ← 8px padding снизу
└─────────────────────────┘
```
